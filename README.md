# 🚗 Ride Booking Analysis Dashboard

## 📊 Project Overview
This project analyzes ride booking data to identify patterns in cancellations, revenue, and ride status.

## 🎯 Objective
- Analyze cancellation trends
- Identify peak cancellation time slots
- Understand driver vs customer cancellations
- Improve business decision-making

## 🛠 Tools Used
- Excel (Data Cleaning)
- Power BI (Dashboard & Visualization)
- DAX (Measures & Calculations)

## 📌 Key Insights
- Evening time shows highest cancellations
- Driver cancellations are higher than customer cancellations
- Majority rides are successfully completed
- Certain time slots generate higher revenue
